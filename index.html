Complete System Analysis

I developed a report detailing the implementation of an IT system for a fictional insurance company, TechTrust, aiming to establish its operations in Portugal. The project includes an overview of the company, its structure, employees, and departments, along with the setup of the physical system, required equipment, and office layout.

The report also covers the planning and scheduling of the network installation and the development of policy management software. Using virtual machines, I demonstrated the server configuration as a domain controller, IP planning, and user permissions. Additionally, I conducted a system analysis with an event list, context diagram, and use case diagrams.

To ensure a well-structured system, I performed functional analysis with Level 0 and lower-level diagrams, along with data normalization using a data dictionary. Finally, I designed the entity-relationship diagram, table schema, and class diagram to define the system's architecture.
